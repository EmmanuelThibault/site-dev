Site web

J'ai réalisé ce projet web pour des clients kinésithérapeutes travaillant sur le domaine respiratoire.

L'objectif ? Permettre le suivi des patients à travers une application web en référençant leurs évaluations diagnostiques dans le but de rendre compte plus efficacement des avancées de chacun suivant le protocole, le programme défini.

En collaboration avec un développeur expert, je devais réaliser les maquettes prédéfinies avec les kinésithérapeutes en manipulant les modèles de données construient au préalable.

Le travail que j'ai réalisé a donné satisfaction à mon client et cela m'a permis de construire grandement mes compétences.

Technologies utilisées : HTML, CSS, PYTHON, DJANGO, DOCKER
